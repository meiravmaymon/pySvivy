# סיווג סעיפים לדיון במועצת העיר

מסמך זה מגדיר את הסיווגים האפשריים לסעיפים המועלים לשולחן מועצת העיר, בהתאם לפקודת העיריות והתקנון לישיבות מועצה.

---

## קטגוריות ראשיות

### 1. תקציב ומימון (BUDGET)
| קוד | שם הסיווג | תיאור | דוגמאות |
|-----|-----------|-------|---------|
| `BUDGET_ANNUAL` | תקציב שנתי | אישור התקציב השנתי הכולל | תקציב רגיל לשנת 2025 |
| `BUDGET_TABAR` | תב"ר | תקציב בלתי רגיל לפרויקטים | תב"ר לשיפוץ גני ילדים |
| `BUDGET_RESERVE` | תקציב מילואים | אישור תקציב לשינויים דחופים | תוספת תקציב למחלקת רווחה |
| `BUDGET_TRANSFER` | העברה תקציבית | העברת כספים בין סעיפים | העברה מתקציב פארקים לחינוך |

## מילות מפתח לזיהוי אוטומטי

### תקציב ומימון
```
תקציב, תב"ר, תב"ר, העברה תקציבית, מילואים, הקצאה, מימון, הכנסות, הוצאות
```

### חוזים
```
חוזה, מכרז, התקשרות, הסכם, ספק, קבלן, פטור ממכרז, ועדת מכרזים
```

### מינויים
```
מינוי, מבקר, גזבר, מנהל, נציג, ועדה, דירקטוריון, חבר מועצה
```

### חוקי עזר
```
חוק עזר, תקנה, אגרה, היטל, ארנונה, תעריף, קנס
```

### נכסים
```
מקרקעין, נכס, קרקע, מגרש, מבנה, חכירה, השכרה, מכירה, רכישה, הפקעה, שעבוד
```

### הלוואות
```
הלוואה, ערבות, אשראי, השקעה, פיקדון, התחייבות
```

### תאגידים
```
תאגיד, חברה עירונית, עמותה, חברה בת, דירקטוריון
```

### תכנון
```
תכנית, תב"ע, בנייה, מתאר, הקלה, שימוש חורג, היתר
```

### שמות
```
הנצחה, קריאה על שם, שם רחוב, שם גן, כיכר, אנדרטה
```

### דוחות
```
דוח, ביקורת, רו"ח, כספי, רבעוני, שנתי
```

### עדכונים
```
דבר ראש העיר, שאילתה, דיווח, עדכון, הודעה
```

### פרוטוקולים
```
פרוטוקול, אישור החלטות, ועד הנהלה
```

## שימוש במערכת

1. **זיהוי אוטומטי**: המערכת מנתחת את כותרת הסעיף ותוכנו ומציעה סיווג
2. **אישור ידני**: המשתמש יכול לאשר או לשנות את הסיווג המוצע
3. **למידה**: המערכת לומדת מהתיקונים ומשפרת את הזיהוי האוטומטי
